# Name Ihres Spiels

## Das Spiel
- Kurze Beschreibung des Spiels:

Unsere Spielidee heißt **Die Geschichte eines Drachleins**. Der Hauptcharakter ist **Gosha**, ein kleiner Drache, der seine vier Freunde verloren hat. Jeder Freund steht für eine Jahreszeit: Winter, Frühling, Sommer und Herbst. Gosha reist durch vier Level, die jeweils einer Saison gewidmet sind, um sie zu finden. Unterwegs sammelt er wichtige Gegenstände, die zu jedem Freund passen. Nachdem er einen Freund gerettet hat, wird der Skin dieses Freundes im Spiel freigeschaltet. Das Ziel ist, alle 4 Freunden zu finden.

- Wie wird das Spiel bedient? (Tastaturbelegung)

Für das Spiel werden eine **Maus** und eine funktionierende **Tastatur** benötigt. Die Tasten **links**, **rechts** und **oben** dienen zur Bewegung, während die **Escape-Taste** und die Maus zur Navigation in der Benutzer-UI dienen.

## Installation
Anleitung, wie das Spiel geladen und gestartet wird:
1. Voraussetzungen: Python 3.10+, Arcade 3.1+
2. Laden Sie das Repository herunter.
3. Öffnen Sie dann ein Terminal und wechseln Sie in das Verzeichnis `geschichte-der-drachlein`.
4. Starten Sie das Spiel mit `python game.py`
